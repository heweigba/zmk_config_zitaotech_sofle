# zmk_config_zitaotech_sofle
zmk configuration for zitaotech sofle split keyboard

# 功能说明

## 1. 保留原Minilaair的习惯
把大拇指自然放的那个键设置为fn，在fn加sedf是上下左右，fn加j是退格，fn加空格是回车，其他还未设置。

## 2. 右手小红点双模式
- **默认层**：小红点控制鼠标指针移动
- **鼠标层**（按住H键）：小红点变成滚动模式，上下左右滚动页面
  - 垂直滚动：上下推动滚动页面
  - 水平滚动：左右推动水平滚动

移动小红点时自动进入鼠标层，拇指键变成鼠标左右键，停止移动400ms后自动退出。

## 3. 左手黑莓轨迹球方向键模式
左手轨迹球默认作为方向键使用：上/下/左/右移动光标，方便选取文字。
- **默认**：轨迹球作为方向键
- **Space按住**：轨迹球变为滚动模式

## 4. 编码器功能
- **左编码器**：音量调节（默认层）/ 翻页键（鼠标层）
- **右编码器**：上下方向键

## 5. 后续优化方向
- 轨迹球灵敏度可调
- 不同层不同功能模式
