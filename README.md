# zmk_config_zitaotech_sofle
zmk configuration for zitaotech sofle split keyboard
# 改了什么
## 1. 保留原Minilaair的习惯
把大拇指自然放的那个键设置为fn，在fn加sedf是上下左右，fn加j是退格，fn加空格是回车，其他还未设置。
## 2. 参考群友业余专家的方案，把操作小红点默认改为滚动，长按切层后移动。
长按H进入鼠标层，再把大拇指两个键变成左右键。因为中指按住H 食指去操作小红点的时候，食指有个支点，避免悬空会舒服点。
## 3. 左手黑莓轨迹球默认方向键模式（类似黑莓手机）
左手轨迹球默认作为方向键使用：上/下/左/右移动光标，方便选取文字。
- **默认**：轨迹球作为方向键（每转约触发4-5次方向键）
- **Space按住**：轨迹球变为滚动模式（翻页/滚动）

优化参数：
- 消抖时间80ms，防止连续触发过快
- 按键按住35ms，模拟真实按键手感
- 脉冲触发，线性单步移动

## 4. 后续优化方向
- 轨迹球灵敏度可调
- 不同层不同功能模式
