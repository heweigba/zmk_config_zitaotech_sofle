# zmk_config_zitaotech_sofle
zmk configuration for zitaotech sofle split keyboard
# 改了什么
## 1. 保留原Minilaair的习惯
把大拇指自然放的那个键设置为fn，在fn加sedf是上下左右，fn加j是退格，fn加空格是回车，其他还未设置。
## 2. 右手小红点自动进入鼠标层
- **默认**：小红点直接控制鼠标移动
- **检测到移动时**：自动进入鼠标层，拇指键变成左右键
- **停止移动400ms后**：自动退出鼠标层

不再需要长按H键，操作更自然：移动小红点 → 自动进鼠标层 → 拇指按键点击。
## 3. 左手黑莓轨迹球默认方向键模式（类似黑莓手机）
左手轨迹球默认作为方向键使用：上/下/左/右移动光标，方便选取文字。
- **默认**：轨迹球作为方向键（每转约触发4-5次方向键）
- **Space按住**：轨迹球变为滚动模式（翻页/滚动）

优化参数：
- 消抖时间80ms，防止连续触发过快
- 按键按住35ms，模拟真实按键手感
- 脉冲触发，线性单步移动

## 4. 右手小红点双模式（2025-02-10更新）
- **默认层**：小红点控制鼠标指针移动
- **鼠标层**：小红点变成滚动模式，上下左右滚动页面（类似触摸板双指滑动）
  - 垂直滚动：上下推动滚动页面
  - 水平滚动：左右推动水平滚动
  - 滚动阈值8，避免过于灵敏

## 5. 右手编码器改为鼠标滚轮（2025-02-10更新）
- 右侧旋钮从原来的左右键改为鼠标滚轮
- **默认层**：音量调节（左编码器）+ 鼠标滚轮（右编码器）
- **鼠标层**：翻页键（左编码器）+ 鼠标滚轮（右编码器）

## 6. 后续优化方向
- 轨迹球灵敏度可调
- 不同层不同功能模式
